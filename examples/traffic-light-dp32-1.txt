\ Forgalomiranyito jelzolampa demo DP32-es kartyahoz.
\ A kartyan levo LED-eket hasznalja sajnos mindegyik zold.
\ A sebesseg a kartyan levo potenciometerrel allithato.


decimal
free dup u.
8  constant POT
14 constant PIROS-LED
13 constant SARGA-LED
12 constant ZOLD-LED


: alaphelyzet
  1 PIROS-LED pm!
  1 SARGA-LED pm!
  1 ZOLD-LED  pm!
  0 PIROS-LED d!
  0 SARGA-LED d!
  0 ZOLD-LED  d!
;

: befejezes
  0 PIROS-LED d!
  0 SARGA-LED d!
  0 ZOLD-LED  d!
  0 PIROS-LED pm!
  0 SARGA-LED pm!
  0 ZOLD-LED  pm!
;

: zold-jelzes
  0 PIROS-LED d!
  0 SARGA-LED d!
  1 ZOLD-LED  d!
;

: sarga-jelzes
  0 PIROS-LED d!
  1 SARGA-LED d!
  0 ZOLD-LED  d!
;

: piros-jelzes
  1 PIROS-LED d!
  0 SARGA-LED d!
  0 ZOLD-LED  d!
;

: piros-sarga-jelzes
  1 PIROS-LED d!
  1 SARGA-LED d!
  0 ZOLD-LED  d!
;

: rendorlampa
  alaphelyzet
  begin
    zold-jelzes        10 POT a@ * delayms
    sarga-jelzes        3 POT a@ * delayms
    piros-jelzes       10 POT a@ * delayms
    piros-sarga-jelzes  3 POT a@ * delayms
  ?key until
  befejezes
;

free dup u.
." Felhasznalt memoria:" - . cr 

\ \ Set traffic to autostart
\ ' rendorlampa bootword !

\ \ save to flash
\ syssave
